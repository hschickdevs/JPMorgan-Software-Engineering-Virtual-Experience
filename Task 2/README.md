# JPMC Task 2

### Setup Instructions:

**Requirements:**
* Node >= 18.0.0
* Python3

**Setup:**
1. ```bash
   cd 'Task 2' 
   ```
2. ```bash
   npm install 
   ```
3. ```bash
   npm start 
   ```
4. Open a new terminal and run the following command:
   ```bash
   pip install -r requirements.txt
   ```
5. ```bash
   python3 server.py 
   ```
